# Health-Tracker
An application to track health data.

---

The application is AI generated since June 2026 as an experiment!

---

The Health-Tracker application allows the user to track different health values and provide them to charts and reports. The value which are tracked contain:
* Alcohol consumption
* Blood pressure measurement
* Body temperature measurement
* Medication
* Nutrition (meat, fish veggie)
* Weight measurement

Additionally it is possible to add general comments about health.
